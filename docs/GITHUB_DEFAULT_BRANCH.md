# GitHub 기본 브랜치 변경 가이드

## 📍 Default Branch 설정 위치

GitHub에서 기본 브랜치를 변경하려면 **"Branches"가 아닌 "General" 설정**으로 이동해야 합니다.

---

## 🔧 변경 방법

### 1단계: General 설정으로 이동

1. GitHub 저장소 페이지에서
2. 오른쪽 상단의 **"Settings"** 탭 클릭
3. 왼쪽 사이드바에서 **"General"** 클릭 (맨 위에 있습니다)

### 2단계: Default branch 섹션 찾기

1. General 페이지에서 아래로 스크롤
2. **"Default branch"** 섹션 찾기
   - "Repository name" 섹션 아래에 있습니다
   - 또는 페이지 하단 근처에 있습니다

### 3단계: 브랜치 변경

1. "Default branch" 섹션에서 현재 브랜치 옆의 **스위치/연필 아이콘** 클릭
   - 현재: `window/12.25`
   - 변경: `main` 선택
2. **"Update"** 또는 **"I understand, update the default branch"** 클릭
3. 확인 완료

---

## 🎯 빠른 경로

```
GitHub 저장소 → Settings → General → Default branch 섹션
```

---

## 📸 참고

- "Branches" 페이지는 **브랜치 보호 규칙(Branch Protection Rules)** 설정하는 곳입니다
- "Default branch"는 **"General"** 설정 페이지에 있습니다

---

**팁**: 변경 후 저장소 메인 페이지의 브랜치 선택 드롭다운이 `main`으로 바뀌었는지 확인하세요!
